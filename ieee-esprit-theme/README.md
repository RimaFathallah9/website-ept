## ðŸ“¦ Installation in WordPress

To use this theme on your WordPress site:

### Method 1: Direct Copy (Localhost)
1.  Copy this entire folder `ieee-esprit-theme`.
2.  Navigate to your WordPress installation: `wp-content/themes/`.
3.  Paste the folder there.

### Method 2: Upload via Admin
1.  Compress this folder into a ZIP file (e.g., `ieee-esprit-theme.zip`).
2.  Log in to your WordPress Admin Dashboard.
3.  Go to **Appearance > Themes**.
4.  Click **Add New** > **Upload Theme**.
5.  Select your ZIP file and click **Install Now**.
